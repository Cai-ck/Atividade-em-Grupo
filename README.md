## Sobre o projeto
Desenvolver um dashboard simples usando Streamlit em Python, utilizando as Branchs para separação de tarefas pros membros da equipe.

O código tem o objetivo de ler o arquivo csv e representar os dados de forma visual pro usuário, com a criação de uma tabela e dois tipos de gráficos.
